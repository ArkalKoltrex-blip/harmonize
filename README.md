# Harmonize — Revestimentos & Decoração

Site institucional (demo) da **Harmonize**, especializada em pisos vinílicos,
ripados e revestimentos no sul e sudoeste da Bahia.

> *Acabamento e sofisticação único na região.*

## ✨ Funcionalidades

- **Hero** em banner arredondado com imagem e parallax leve
- **Catálogo de produtos** com filtro funcional por cor e faixa de preço
- Seção de **questionário** (CTA), **Quem Somos**, **Fornecedores** e contato
- Navbar de vidro (glassmorphism) que acompanha todo o site
- Animações com [Motion](https://motion.dev) (scroll reveals, hover, parallax)
- Responsivo e com suporte a `prefers-reduced-motion`

## 🎨 Design System

- **Cores:** paleta creme / espresso / areia (ver [DESIGN.md](DESIGN.md))
- **Tipografia:** Syne (display) + Nunito Sans (texto)
- Tokens em [styles/tokens.css](styles/tokens.css)

## 🛠️ Stack

- React 18 + Vite
- Motion (animações)
- CSS puro com design tokens

## 🚀 Como rodar

```bash
npm install
npm run dev      # ambiente de desenvolvimento (http://localhost:5173)
npm run build    # build de produção
npm run preview  # pré-visualizar o build
```

## 📁 Estrutura

```
harmonize/
├── public/images/      # imagens (hero, ambientes, produtos, logo)
├── src/
│   ├── components/      # Navbar, Hero, Products, Quiz, About, Suppliers, Footer
│   ├── data/           # catálogo de produtos
│   └── index.css       # estilos globais
├── styles/tokens.css   # design tokens
├── DESIGN.md           # sistema de design completo
└── ASSETS.md           # mapa de imagens
```
